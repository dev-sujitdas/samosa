🎲 Roll With Samosa

A modern, interactive website for a boardgame café — Roll With Samosa.
Built with performance, smooth animations, and great UX in mind, this project showcases a rich frontend experience along with an integrated appointment booking system.

🚀 Live Website

### 🌐 https://rollwithsamosa.com

## 📌 Features
🎮 Boardgames Collection
150+ boardgames listed
Filter and sort functionality
Smooth browsing experience
Clean and responsive UI
📅 Appointment Booking
Integrated Odoo Appointment System
Embedded using iframe
Easy table / session booking
✨ Modern UI/UX
Smooth scrolling using Locomotive JS
Animations powered by Framer Motion
Fully responsive design
Clean and engaging layout
🛠️ Tech Stack
Frontend: React.js
Styling: Tailwind CSS
Animations: Framer Motion
Smooth Scroll: Locomotive Scroll
Booking System: Odoo (iframe integration)
